# fuzzer

fuzzer

## Requirements

- pip3 install mechanicalsoup

## Usage

usage: python3 fuzz.py discover/test [-h] [--custom-auth [CUSTOM_AUTH]] url

positional arguments:
  url

optional arguments:
  -h, --help,
  --custom-auth [CUSTOM_AUTH]
  --common-words [COMMON_WORDS]
  --vectors [VECTORS]
  --sensitive [SENSITIVE]
  --timeout=[milliseconds]
